## Proyectos Destacados

### exploraR — Plataforma de Orientación Vocacional con VR
> Sep 2025 – Dic 2025

Aplicación móvil multiplataforma con realidad virtual inmersiva para orientación vocacional universitaria. **Implementada como piloto institucional en CUOH con ~100 usuarios activos.**

- Tours virtuales 360° y experiencias inmersivas con Three.js
- Backend REST API con Node.js, Express y MongoDB Atlas
- Panel administrativo web y dashboard analítico de métricas de uso

`React Native` `Expo` `Node.js` `Express` `MongoDB Atlas` `Three.js` `Cloudinary`

---

### UNINEST — Plataforma de Alojamiento Universitario Seguro
> Sep – Dic 2024

Plataforma web para búsqueda segura de alojamiento universitario con geolocalización. Incluye sistema de control de acceso físico mediante tarjetas RFID integrado con la web.

- Autenticación, gestión de usuarios y listados verificados
- Integración hardware-software con módulo RFID

`React Native` `Node.js` `MongoDB` `Google Maps API` `RFID` `REST API`

---

### SafeCheck School — Control de Accesos con Detección de Alcohol
> Ene – May 2024

Sistema web y móvil basado en IoT para control de accesos escolares con detección de alcohol en tiempo real mediante hardware.

- Integración Arduino + sensores MQ para detección automática
- Identificación de usuarios mediante códigos QR
- Panel administrativo con historial de registros y alertas

`React Native` `Node.js` `MongoDB` `Arduino` `Sensores MQ` `QR`

---

### GeoMap — Visualización Geoespacial de Eventos Municipales
> Hackathon · Dic 2025

Interfaz web responsiva para gestión y visualización geoespacial de eventos catastrales municipales. Primer lugar / finalista en hackathon UTEC.

`HTML5` `CSS3` `JavaScript` `Bootstrap` `MongoDB` `APIs de Mapas`

---

### RECOU — Reciclaje con Impacto Social
> Hackathon · Nov 2023

Aplicación web para localizar centros de donación de PET y aluminio destinados a la fabricación de prótesis. Enfoque social y ambiental.

`JavaScript` `PHP` `Python` `SQLite` `Bootstrap`
